# 🎬 MovieApp (Jetpack Compose)

A simple Android Movie App built using Jetpack Compose.  
This project demonstrates UI design, search functionality, and navigation between screens.

---

## 🚀 Features

- Display a list of movies
- Search and filter movies
- Detail screen for selected movie
- Navigation using movie ID

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Navigation Component

---

## 📱 Screens

- Home Screen  
- Search Screen  
- Detail Screen  

---

## 📌 What I Learned

- Building UI with Jetpack Compose
- Passing data between screens (Navigation)
- Filtering lists based on user input
- Handling nullable data safely

---
